User Interface

Prerequisites
Before you begin, ensure you have met the following requirements:

1. Python 3.x installed.
2. A virtual environment is recommended but not mandatory.

Dependencies
This project uses the following libraries:

tkinter - For building the graphical user interface.
PIL (Python Imaging Library) - Enables image processing capabilities in the Python interpreter.
serial - To encapsulate access to serial ports.
threading - Provides a way to create and synchronize threads.
webbrowser - To allow the display of Web-based documents to users.
numpy - For numerical operations and data manipulation.
pandas - For data analysis and manipulation.
matplotlib - For creating visualizations and plots.
scipy - Used for scientific computing and technical computing.
csv - For reading and writing tabular data in CSV format.
re - Provides regular expression matching operations.
os - Provides a way of using operating system dependent functionality like reading or writing to the file system.
